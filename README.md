# *Программная поддержка подбора персонала на предприятии*
## Recruitment Fullstack App

---

## Описание
*Программная поддержка подбора персонала на предприятии* — это веб-приложение для автоматизации процессов рекрутинга: публикации вакансий, сбора и хранения резюме, планирования и проведения собеседований, общения с кандидатами и генерации отчётов. Система поддерживает аналитические отчёты, экспорт в Excel/PDF, календарь собеседований и механизм расчёта эффективности HR.

## Функционал
- Регистрация и авторизация пользователей (роли: админ / HR / кандидат)
- JWT-аутентификация (токен хранится в `localStorage`)
- CRUD для: вакансий, кандидатов, HR-профилей, резюме, собеседований, уведомлений
- Календарь собеседований (создание/редактирование/удаление)
- Чат с автоматическими системными сообщениями (при откликах/приглашениях)
- Модальные окна для детального просмотра сущностей
- Экспорт/импорт: Excel (XLSX), PDF (формирование отчётов по собеседованиям)
- Графики и визуализация статистики по вакансиям и откликам
- Расчёт лучшего HR по метрикам (время закрытия вакансии, конверсия и т.п.)
- Пагинация, фильтрация, сортировка списков
- Интеграция с внешним API (Нацбанка) для конвертации валют

## Реализация и архитектура
Приложение построено по клиент-серверной модели (SPA ↔ REST API).  
Основные слои на сервере: Controller → Service → Repository → Database.  
Фронтенд — SPA на React, использует маршрутизацию, формы с валидацией и локальное хранение токена.

Ключевые моменты:
- Модульность — отдельные модули для вакансий, кандидатов, собеседований, отчетности.
- Асинхронные действия выполняются через REST API с обработкой ошибок и статусами.
- Безопасность: хранение паролей в виде хешей, JWT, проверки ролей на сервере.
- Отчёты генерируются на сервере (PDF) и/или на клиенте (экспорт Excel).

## API и интеграции

### REST API  

**Основные endpoint-ы:**

- **POST `/api/auth/register`** — регистрация  
- **POST `/api/auth/login`** — вход (возвращает JWT)  
- **GET `/api/vacancies`** — список вакансий (фильтры, пагинация)  
- **POST `/api/vacancies`** — создать вакансию (HR/админ)  
- **GET `/api/candidates`** — список кандидатов  
- **POST `/api/resumes`** — загрузка резюме  
- **POST `/api/interviews`** — создать собеседование  
- **GET `/api/reports/interview/{id}`** — PDF-отчёт по собеседованию  
- **GET `/api/reports/export?format=xlsx`** — экспорт аналитики в Excel  

Интеграция с **API Нацбанка** используется для получения курсов валют и пересчёта вознаграждений

## Особенности реализации

- **Модальные окна** — быстрый просмотр кандидатов и вакансий без перехода на отдельную страницу  
- **Формы** — React Hook Form + Yup (валидация, UX, вывод ошибок)  
- **Авторизация** — хранение JWT в `localStorage`, axios-интерсепторы для `Authorization`  
- **Графики и аналитика** — визуализация KPI: время закрытия вакансий, источники откликов и др.  
- **Отчёты**  
  - PDF-отчёты: участники интервью, оценки, заметки  
  - Excel-экспорт: аналитика по вакансиям и процессу найма

## Архитектурные паттерны

- **Strategy** — статистика по собеседованиям
- **Builder** — модель резюме  
- **Observer** — система уведомлений (отклики, приглашения, изменение статуса)
